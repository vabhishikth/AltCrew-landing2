# AltCrew - Fitness Community Platform

AltCrew is a high-energy, brutalist-designed landing page for a modern fitness community platform. It connects runners, hybrid athletes, and wellness enthusiasts with local crews, while providing organizers with the tools they need to manage events, payments, and guestlists.

- **Brutalist Kinetic Design**: High-contrast, acid-lime aesthetics with aggressive typography (Oswald & Space Grotesk).
- **Interactive Discovery**: Horizontal scrolling marquees and bento-grid layouts.
- **Host Tools Visualization**: Dashboard previews for event management, payments, and analytics.
- **Community Proof**: Testimonials and media tickers to build trust.
- **Responsive**: Fully optimized for mobile and desktop.

## üåê How to Share (Get a Public URL)

### Option 1: Deploy to Vercel (Recommended for Public Sharing)
This provides a permanent, high-performance URL (e.g., `altcrew.vercel.app`).

1. Install Vercel CLI: `npm i -g vercel`
2. Run command: `vercel`
3. Follow the prompts (press Enter).
4. Share the **Production** URL provided.

### Option 2: Local Tunnel (For sharing on your network)
If you want to share your `localhost:3000` temporarily:

1. Start your app: `npm start`
2. In a new terminal, run: `npx ngrok http 3000`
3. Share the `https://....ngrok-free.app` link.
